# Backgrounds

Place your background images in this folder.

## Naming Convention

Each background image must be named **exactly** with its `backgroundId`:

```
1.jpg
2.jpg
studio.jpg
beach.png
gradient-purple.webp
```

The `backgroundId` used in the web form must match the filename (without extension).

## Supported Formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Recommended Size
- **1920 × 1080 px** (Full HD) or **2000 × 2000 px** (square)
- The person photo is scaled to fit within the background height preserving aspect ratio, then centered.
